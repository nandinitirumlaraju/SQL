# Stored Procedures

## Description

This project demonstrates the use of a stored procedure to calculate yearly sales for one or more customers.

## Procedure Name

get_monthly_gross_sales_for_customer

## Input

Customer code(s)

Example:

90002002,90002008

## SQL Concepts Used

- CREATE PROCEDURE
- Input Parameters
- FIND_IN_SET()
- INNER JOIN
- SUM()
- ROUND()
- GROUP BY
- ORDER BY
- User Defined Function

## Output

Returns:

- Fiscal Year
- Yearly Sales
