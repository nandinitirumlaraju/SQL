# Monthly Product Sales Report

## Description
This query retrieves monthly product sales information by joining:

- fact_sales_monthly
- dim_product
- fact_gross_price

## SQL Concepts Used
- INNER JOIN
- User Defined Function
- WHERE clause
- ORDER BY
- ROUND()

## Output Columns
- Date
- Product Code
- Product
- Variant
- Sold Quantity
- Gross Price
- Gross Price Total
